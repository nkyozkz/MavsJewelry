# MavsJewelry

MavsJewelry es un sitio web desarrollado para un emprendimiento de joyería artesanal, diseñado para exhibir productos de forma clara y atractiva a modo de catalogo, facilitando que los clientes conozcan las distintas colecciones y puedan contactar con el negocio.

## Deploy
https://mavsjewelry.netlify.app

## Descripción

El objetivo del proyecto es ofrecer una presencia web para el emprendimiento, permitiendo mostrar las piezas de joyería de manera organizada mediante una interfaz sencilla y adaptable a distintos dispositivos.

## Características

- Catálogo de productos.
- Diseño responsivo para dispositivos móviles y escritorio.
- Interfaz limpia y fácil de navegar.
- Organización de productos por secciones.
- Información de contacto del emprendimiento.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Boostrap
